
A cross-platform E-commerce application built with .NET MAUI Blazor Hybrid.

## Tech Stack
- **Frontend:** Blazor Hybrid (Web & Android)
- **Backend:** ASP.NET Core Web API
- **Database:** Entity Framework Core & SQL Server
- **Authentication:** Custom Auth Service

## Key Features
- Neon Glossy UI/UX
- Real-time Sales Dashboard
- Order confirmation emails via SMTP
- Admin panel to manage products

## How to Run Locally

Follow these steps to get a local copy of GlossyCommerce up and running.

### 1. Prerequisites
Ensure you have the following installed on your machine:
- [.NET 8.0 SDK](https://dotnet.microsoft.com/download) (or latest)
- [Visual Studio 2022](https://visualstudio.microsoft.com/) (with **.NET Multi-platform App UI development** and **ASP.NET and web development** workloads checked)
- [SQL Server Management Studio (SSMS)](https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms)
